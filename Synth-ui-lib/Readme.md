# Synth UI

Modern React UI component library with beautiful, reusable, and animated components built for fast development.

---

## ? Features

- ? Lightweight and fast
- ?? Modern UI components
- ?? Reusable architecture
- ?? Easy npm installation
- ?? Built for React 18+
- ?? TypeScript support
- ?? Developer-friendly API

---

## ?? Installation

```bash
npm i @hustlerx99/synth-ui
```

---

## ?? Usage

```jsx
import { Button } from "@hustlerx99/synth-ui";

export default function App() {
  return (
    <div>
      <Button>Click Me</Button>
    </div>
  );
}
```

---

## ??? Development

Clone the repository:

```bash
git clone https://github.com/hustlerx99/synth-ui.git
```

Install dependencies:

```bash
npm install
```

Run development build:

```bash
npm run dev
```

Build library:

```bash
npm run build
```
